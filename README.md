
# 🧹 Projeto de Refatoração — Código Limpo e Boas Práticas

## 📌 Descrição

Este projeto tem como objetivo aplicar os princípios do **Código Limpo (Clean Code)** na refatoração de um código legado. Esta atividade faz parte do trabalho acadêmico da disciplina de **Engenharia de Software**, e visa melhorar a **legibilidade**, **manutenibilidade** e **eficiência** do código, **sem alterar sua funcionalidade**.

## 🔨 Tecnologias Utilizadas

- Python 3.x  
- Biblioteca padrão (`math`)  
- `unittest` (para testes unitários)  

## 🎯 Objetivos Atendidos

- Aplicação dos princípios **SOLID**, **DRY**, **KISS** e **YAGNI**  
- Implementação do **Design Pattern Factory**  
- Criação de **testes unitários**  
- Organização do código seguindo o paradigma de **programação orientada a objetos**  

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/isabelyferreira/refatoracao-clean-code.git
```

2. Execute o código:

```bash
python formas.py
```

3. Execute os testes unitários:

```bash
python -m unittest test_formas.py
```
